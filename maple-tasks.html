One-time setup

To sync tasks with Google Calendar, you'll need to create a free Google Cloud project. Takes about 5 minutes.

Step 1 — Create OAuth credentials

  1. Go to console.cloud.google.com and create a project (name it "Maple Tasks")
  2. Open Calendar API library → click Enable
  3. Open OAuth consent screen → choose External, fill in name/email, save
  4. Open Credentials → + Create Credentials → OAuth Client ID → Application type: Web application
  5. Under "Authorized JavaScript origins", add the URL where you'll host this file (e.g., https://yourname.github.io or http://localhost:8000 for testing)
  6. Copy the Client ID (looks like 123456789-abc.apps.googleusercontent.com)

Step 2 — Paste your Client ID below

Step 3 — Calendar to sync to (optional)

Leave blank for primary calendar, or paste a calendar ID (settings → integrate calendar)

Tip — share with your dad

Create a shared Google Calendar first (e.g., "Maple MPSS Tasks"), share it with both your accounts with edit permissions, then paste its calendar ID above. Both of you will see and edit the same tasks live.
