Welcome to Maple MPSS

One-time setup. You'll need a Google Cloud OAuth Client ID and a Google Sheet to use as your database.

How do I get these?
  1. In Google Cloud Console, create an OAuth Client ID (Web application).
  2. Add this site's URL to "Authorized JavaScript origins" (no trailing slash).
  3. Enable both Google Sheets API and Google Calendar API.
  4. Create a Google Sheet, run the setup script (provided separately), copy the Sheet ID from the URL.
  5. Add yourself + Dad as Test Users in the OAuth consent screen.
